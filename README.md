# smart-chess-academy-system

A chess academy management system built with React, TypeScript, Vite, Tailwind CSS, shadcn-ui, and Supabase.

## Tech stack

- Vite + React 18 + TypeScript
- Tailwind CSS + shadcn-ui
- TanStack React Query v5
- React Hook Form + Zod
- Supabase (Auth, PostgreSQL, RLS)
- Recharts

## Getting started

```sh
git clone <YOUR_GIT_URL>
cd smart-chess-academy-system
npm install
npm run dev
```

Copy `.env` and fill in your Supabase project URL and anon key.
